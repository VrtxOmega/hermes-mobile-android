# Release Notes

## 0.1.0

Initial sanitized Android client export.

Included:

- Kotlin/Compose app shell
- dashboard password-login flow
- encrypted cookie persistence
- session list and session resume
- gateway WebSocket client
- streaming message handling
- interaction rendering for approval, clarify, sudo, and secret requests
- model/profile surfaces
- management readbacks
- Workbench launch modes
- embedded browser controls
- source-aware artifact/evidence parsing
- send safety and retry layer
- focused unit tests

Not included:

- release signing
- production distribution
- public OAuth flow
- biometric app lock
- notifications
- destructive/admin controls
